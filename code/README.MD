# Code Documentation

## Overview

This directory contains all analysis scripts and functions as a template.

## Dependencies

- Python 3.9+
- pandas==2.0.0
- numpy==1.21.0
- matplotlib==3.4.3
- seaborn==0.11.2
- scikit-learn==1.0.2

## Setup Instructions

1. Create virtual environment:

```bash
python -m venv env
source env/bin/activate  # On Windows: env\Scripts\activate
```
